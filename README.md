Fall Of Ferald
==============

## Overview
SRPG based on the SRPG_Engine repo that was developed prior.
Currently in the prototype stage.

## Completed Features
* Rudimentary levels
* Tooltips
* Dialogue Boxes
* Units
* Super Basic AI
* Create a 2D camera so that level sizes aren't restricted to the size of the window
